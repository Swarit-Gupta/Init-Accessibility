🧠 Neurodivergent-Friendly

Interfaces and settings tailored for users with ADHD, dyslexia, autism, anxiety, and other neurological differences.

📖

Dyslexia-Friendly Font

Switch the entire page to a font with weighted, distinct letterforms designed to reduce character reversal for dyslexic readers.

🚫

Reduced Animations

Disable all non-essential transitions and animations to prevent distraction or discomfort for users sensitive to motion.

↕️

Adjustable Line Spacing

Increase line height to make text easier to track across a page — particularly helpful for users with dyslexia or ADHD.

🎯

Clear Focus Indicators

All interactive elements display a prominent focus ring when navigated by keyboard, supporting users who rely on visual cues.

🎨

Color-Vision Support

Select a color-blind simulation mode (protanopia, deuteranopia, or tritanopia) to preview and adapt the palette for your vision type.

🧘

Minimal, Predictable Layout

Consistent layout, clear headings, and plain language help users with executive function challenges navigate without cognitive overload.
